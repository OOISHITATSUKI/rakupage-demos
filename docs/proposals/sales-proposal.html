よくある質問

Q LINE公式アカウントを持っていなくても利用できますか？ +

はい、可能です。LINE公式アカウントの開設から当社が無料で代行いたします。既存のLINE公式アカウントをお持ちの場合も、そのまま連携できます。

Q AIが間違った予約を取ることはありませんか？ +

AIは院の空き状況をリアルタイムで確認しながら予約を受け付けるため、ダブルブッキングは発生しません。また、管理画面からすべてのやり取りを確認でき、必要に応じて手動で修正・介入することも可能です。

Q 既存の予約システムと併用できますか？ +

はい、電話予約や来院時の直接予約と併用いただけます。管理画面で手動予約の追加も可能ですので、すべての予約を一元管理できます。

Q 患者さんがうまくLINEを使えるか心配です +

LINE AI受付は自然な日本語での会話に対応しているため、特別な操作は不要です。「予約したい」「腰が痛い」など、普段通りのメッセージを送るだけで予約が完了します。ご年配の患者さんでも簡単にお使いいただけます。

Q 解約に縛りはありますか？ +

最低利用期間の縛りはございません。月単位でのご契約となりますので、いつでも解約可能です。14日間の無料トライアル期間中に解約された場合、料金は一切発生しません。

Q 導入にどのくらいの時間がかかりますか？ +

最短で翌営業日から運用を開始いただけます。お申込み後、当社にてLINE連携とメニュー設定を行いますので、お客様の作業はほぼ不要です。

Q 患者さんの個人情報や予約データのセキュリティは大丈夫ですか？ +

はい、万全の体制を取っております。

① 院ごとのデータ完全分離
各院のデータは独立したIDで管理されており、他院のデータとは一切混ざりません。A院の患者情報がB院に見えることは絶対にありません。

② LINEの本人認証
各患者はLINEの固有ユーザーIDで識別されます。このIDは偽装できないため、他人が予約を確認・キャンセルすることは不可能です。予約のキャンセルもLINE本人からのみ行えます。

③ SSL暗号化通信
すべての通信はSSL（HTTPS）で暗号化されており、第三者による傍受を防いでいます。

④ 管理画面のアクセス制御
管理画面は院ごとに固有の認証トークンで保護されており、権限のない第三者はアクセスできません。

Q 他の患者さんに予約を勝手にキャンセルされることはありませんか？ +

ありません。予約のキャンセルはLINEの固有ユーザーIDで本人確認を行っています。キャンセルを要求すると、そのLINEアカウントに紐づいた予約のみが表示され、他の患者さんの予約は一切表示されません。そのため、なりすましや誤キャンセルは技術的に不可能な仕組みになっています。
