本スクリプトの使い方
各院のリサーチ情報に基づき、相手の課題に合わせたカスタマイズトークを用意しています。電話をかける前に該当院のセクションを確認し、相手のペインポイントを把握した上で電話してください。共通パートは全院で使えるベースです。

共通営業ポイント

サービス概要（30秒で伝える）

LINE公式アカウントにAIを搭載して、24時間自動で予約受付ができるシステムです。
患者さんがLINEでメッセージを送ると、AIが症状のヒアリングから空き枠の提案、予約確定まですべて自動で行います。
施術中に電話が鳴って手を止める必要がなくなります。

刺さるキーワード（繰り返し使う）

  • 施術中の電話対応が不要 - 最大のペインポイント
  • 24時間受付 - 夜間・休診日の取りこぼし防止
  • 患者さんの8割はLINEユーザー - 電話よりハードルが低い
  • 新患の離脱防止 - 電話がつながらず他院に流れるのを防ぐ
  • 受付スタッフ不要 - 人件費の削減
  • 初期費用ゼロ・モニター無料 - リスクなく試せる
