秋田の厳しい冬にも耐えうる高品質な家づくりと、お客様との末永いお付き合いを大切にしています。

FAQ

よくある質問

新築の相談は無料ですか？

+
はい、ご相談・お見積りは無料です。まずはお気軽にお電話またはお問い合わせフォームよりご連絡ください。お客様のご要望を伺いながら、最適なプランをご提案いたします。

リフォームだけでもお願いできますか？

+
もちろんです。水回りの改修や内装リフォーム、バリアフリー化など、小規模な工事から大規模リフォームまで幅広く対応しております。他社で建てられた住宅でもお気軽にご相談ください。

工期はどのくらいかかりますか？

+
新築の場合は規模にもよりますが、着工から約4〜6ヶ月程度が目安です。リフォームは内容により数日〜数ヶ月と幅があります。詳しくはお打ち合わせ時にご説明いたします。

対応エリアはどこまでですか？

+
北秋田市を中心に、大館市・能代市・鹿角市など近隣市町村まで対応しております。エリア外の場合でもまずはご相談ください。可能な限り対応いたします。

アフターサービスはどのような内容ですか？

+
お引き渡し後の定期点検をはじめ、不具合の修繕、住まいに関するご相談など、末永くサポートいたします。些細なことでもお気軽にお問い合わせください。建てた後こそ大切なお付き合いと考えております。
